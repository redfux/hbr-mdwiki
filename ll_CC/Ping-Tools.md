# Ping-Tools
## Prettyping

https://denilson.sa.nom.br/prettyping/

    curl -O https://raw.githubusercontent.com/denilsonsa/prettyping/master/prettyping
    chmod +x prettyping
    ./prettyping whatever.host.you.want.to.ping

## noping
https://noping.cc/

    apt install oping
    noping whatever.host.you.want.to.ping
